## Base URL
The base-URL is

```bash
https://www.steamwebapi.com/
```

Easily include your API key in the request by appending it to the URL. Retrieve your API key from your profile for seamless integration. For instance, access the inventory endpoint using this format:

```bash
https://www.steamwebapi.com/steam/api/items?key=$YOUR_API_KEY
```
