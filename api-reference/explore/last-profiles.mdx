---
title: "Last Random profiles"
api: "GET https://www.steamwebapi.com/explore/api/last?key={apikey}&limit={limit}"
description: "This endpoint return random 5 Profiles with Inventory Informations and Items."
---

### Path

<ParamField path="apikey" type="string" required>
  Your API key under: Dashboard (TOP RIGHT)
</ParamField>

<ParamField path="limit" type="number" default="5">
  Limit of the items max. 50 every 1 profile costs 2 credits for example Limit 5 costs 10 credits.
</ParamField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://www.steamwebapi.com/explore/api/last?key=$YOUR_API_KEY&limit=$LIMIT' \
--header 'Content-Type: application/json' \
```
</RequestExample>
