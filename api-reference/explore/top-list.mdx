---
title: "Top list profiles"
api: "GET https://www.steamwebapi.com/explore/api/toplist?key={apikey}&limit={limit}"
description: "This endpoint retrieves the top 5 profiles with the highest combined value across inventory, information, and items."
---

### Path

<ParamField path="apikey" type="string" required>
  Your API key under: Dashboard (TOP RIGHT)
</ParamField>

<ParamField path="limit" type="number" default="5">
  Limit of the items max. 50 every 1 profile costs 2 credits for example Limit 5 costs 10 credits.
</ParamField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://www.steamwebapi.com/explore/api/toplist?key=$YOUR_API_KEY&limit=$LIMIT' \
--header 'Content-Type: application/json' \
```
</RequestExample>
