---
title: "Float"
api: "GET https://www.steamwebapi.com/float/api/item?key={apikey}&inspectlink={inspectlink}&s={s}&m={m}&a={a}d={d}"
description: "This endpoint return a float of skin."
---

You can provide parameters as follows: either inspectlink, or a combination of m, a, d, and s. At least one parameter is required.

### Path

<ParamField path="apikey" type="string" required>
  Your API key under: Dashboard (TOP RIGHT)
</ParamField>

<ParamField path="inspectlink" type="string">
  Inspect Link of the item e.g. steam://...
</ParamField>

<ParamField path="s" type="string">
  Steam ID - not required if you have inspectlink
</ParamField>

<ParamField path="m" type="string">
  Market Item ID - not required if you have inspectlink
</ParamField>

<ParamField path="a" type="string">
  Asset ID - not required if you have inspectlink
</ParamField>

<ParamField path="d" type="string">
  D ID - not required if you have inspectlink
</ParamField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://www.steamwebapi.com/float/api/item?key=$YOUR_API_KEY&inspectlink=$INSPECT_LINK' \
--header 'Content-Type: application/json' \
```
</RequestExample>
