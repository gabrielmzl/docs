---
title: "List all currencies"
api: "GET https://www.steamwebapi.com/currency/api/list?key={apikey}&base={base}"
description: "This endpoint list all currencies."
---

### Path

<ParamField path="apikey" type="string">
  Your API key under: Dashboard (TOP RIGHT)
</ParamField>

<ParamField path="base" type="object">
  The currency code you want to convert from (base currency) for currency conversion. If not provided, USD is used as the base currency.
</ParamField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://www.steamwebapi.com/currency/api/list?key=$YOUR_API_KEY&base=$BASE' \
--header 'Content-Type: application/json' \
```
</RequestExample>
